# 🚀 Personal Portfolio Website

A modern, responsive, and interactive **personal portfolio website** built using **HTML, CSS, and JavaScript**.  
This portfolio showcases my skills, projects, education, and provides a **working contact form** integrated with **EmailJS**.

## ✨ Features

- 🎨 Modern dark UI design
- 📱 Fully responsive (Mobile, Tablet, Desktop)
- 🍔 Hamburger menu for mobile
- 🧭 Smooth scrolling navigation
- 🔦 Active navbar highlight on scroll
- 🎬 Scroll animations (fade & slide)
- 💼 Projects showcase section
- 🎓 Education & timeline section
- 📬 Contact form with **EmailJS integration**
- 🔔 Toast notification + loading spinner
- ⚡ Lightweight & fast performance

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **EmailJS** – Contact form emails
- **Particles.js** – Animated background
- **GitHub Pages** – Deployment
